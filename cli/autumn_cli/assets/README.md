# Assets

This directory contains static assets used by the autumn-cli package.

## Icons

### PNG Format
- `autumn_icon.png` - Main Autumn icon (non-transparent variant)
- `autumn_icon_clear.png` - Clear/transparent variant of the Autumn icon

### ICO Format
- `autumn_icon.ico` - Main Autumn icon (ICO format for Windows)
- `autumn_icon_clear.ico` - Clear/transparent variant (ICO format for Windows)

## Usage

- **macOS**: Uses PNG icons with `terminal-notifier`
- **Windows**: Uses ICO icons with `plyer` and PowerShell toast notifications
- **Linux**: Icon not currently supported (uses `notify-send`)
